# ReactBits-Style Components Integration

Custom UI components inspired by ReactBits have been integrated into the landing page using **Framer Motion** to create premium, interactive animations.

## Components Implemented

### 1. Aurora Background
Soft, animated gradient blobs that create a dynamic "northern lights" effect behind the hero section.

![Hero with Aurora](file:///Users/rae/.gemini/antigravity/brain/dd761c29-f2d3-4daa-87a6-d9cfdb0bc63f/hero_section_1769422221107.png)

---

### 2. BlurText Animation
Letter-by-letter reveal on the main heading with blur-to-sharp transition.

---

### 3. DecryptedText Effect  
Matrix-style scramble animation on the "Blockchain Recovery Layer" pill - characters cycle through random glyphs before revealing final text.

---

### 4. TiltCard 3D Effect
Interactive perspective shift on feature images with subtle glare overlay.

![Feature Card Hover](file:///Users/rae/.gemini/antigravity/brain/dd761c29-f2d3-4daa-87a6-d9cfdb0bc63f/feature_card_hover_1769422329089.png)
