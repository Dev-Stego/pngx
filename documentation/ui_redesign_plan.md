|-------|---------|

## Key Design Principles

1. **Single-column stepped flow** - Easy to follow, no alignment issues
2. **Large clickable cards** for mode selection - Not tiny cramped boxes  
3. **Generous spacing** - 24-32px between sections
4. **Visual step indicators** - Numbered badges
5. **Futuristic glow effects** - Subtle gradients and hover glows
6. **Micro-animations** - Smooth transitions between states
